# Notas legales

> Este documento es informativo y **no constituye asesoría legal**.

## Naturaleza de los datos

- Las **fechas**, los **identificadores normativos** (tipo, número, año de leyes y decretos) y las **referencias legales** son **hechos**. Los hechos no son objeto de derecho de autor.
- El dataset **no reproduce prosa extensa protegida de terceros**: sólo datos estructurados (fechas, nombres breves, clasificación, identificadores de normas y enlaces a fuentes oficiales).

## Origen

El dataset se **genera a partir de fuentes internas del proyecto**, procesadas y normalizadas por el equipo. La trazabilidad técnica se conserva en material interno para reproducibilidad y auditoría.

Ver [`LICENSE`](../LICENSE).

## Irrenunciabilidad

El campo `irrenunciability_category` refleja la categoría declarada en las fuentes para el año correspondiente. Cuando el valor se infiere de un rango (en lugar de un dato por año explícito), se marca como inferencia en la trazabilidad interna (`source.notes`). No debe usarse como única base para decisiones laborales sin verificación.

## Uso responsable

Para cómputo de plazos, vencimientos o efectos laborales, verificar contra las fuentes oficiales vigentes. El proyecto no garantiza idoneidad para un fin específico.
